\# NOVA PROJECT



Sistema turístico basado en QR.

\- qr-backend → API Node.js + PostgreSQL

\- nova\_app → Aplicación móvil Flutter

\- nova\_dashboard → Panel web de administración

\- Registro y login

\- Escaneo de QR

\- Sistema de recompensas

\- Dashboard con analíticas
